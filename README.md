# Restore manjora


Set default browser:

- `xdg-mime default opera.desktop x-scheme-handler/http`
- `xdg-mime default opera.desktop x-scheme-handler/https`
- `xdg-settings set default-web-browser opera.desktop`
- edit the `~/.config/mimeapps.list`
